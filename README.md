# 🌲 Lanzamiento de Caber (Caber Toss)

Este proyecto es una solución algorítmica desarrollada en **Java** para procesar y calificar los lanzamientos de troncos en una competencia de *Caber Toss*. El programa analiza la validez de los lanzamientos basándose en la distancia y el ángulo de caída.

## 📖 Descripción del Problema

En la competencia de Caber, no solo importa la distancia, sino también la precisión del ángulo (representado a menudo como las horas de un reloj). El programa lee un conjunto de lanzamientos desde un archivo, aplica las reglas de calificación y genera un reporte con los resultados procesados.

## 🛠️ Tecnologías Utilizadas

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)

- **Lenguaje**: Java
- **Paradigma**: Programación Orientada a Objetos.
- **Manejo de Datos**: Lectura y escritura de archivos de texto (`.in` y `.out`).

## 📂 Estructura del Proyecto

- `src/`: Carpeta con el código fuente (`.java`).
- `caber.in`: Archivo de entrada con los datos de los lanzadores y sus tiros.
- `caber.out`: Archivo de salida con la clasificación final.

## ⚙️ Ejecución

Para compilar y ejecutar el proyecto, utiliza los siguientes comandos en la terminal:

1. **Compilación:**
   ```bash
   javac src/Caber.java
   ```
2. **Ejecución:**
   ```bash
   java src/Caber  
   ```
Asegurate de que el archivo caber.in se encuentre en el directorio raíz para que el programa pueda procesar los datos correctamente.
